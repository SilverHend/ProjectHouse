# Техническое задание.

0. Размер (Дома **9x15**) + (Террасы **7x4**) 
1. Дом должен быть выполнен в стиле (**Микея-3/Микея-5**)
2. С правой части дома должна быть терраса, вход на которую через кухню.
3. Тип кровли фальцевая.
